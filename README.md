# OdysseusApi
## [Live demo](https://odysseusapi2021.azurewebsites.net/)  
Type three-letter destination code after domain adress in URL bar, for example 'xxxxx.com/PAN'. Available country codes:  
- CAN  
- USA
- MEX
- BLZ
- GTM
- SLV
- HND
- NIC
- CRI
- PAN
